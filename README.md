# Fase-5-Fundamentos-de-programaci-n
Este repositorio contiene el código fuente en Python para el Problema 5 de la evaluación final. El programa procesa una matriz con las horas trabajadas por 4 empleados durante la semana, calcula el total de horas por persona y clasifica la jornada como "Sobretiempo" (si excede las 40 horas) o "Horario Estándar".
